# Emotion-Based Spotify Playlist Recommendation System

## Project Description
This project recommends Spotify playlists based on the user's facial emotions.  
It detects emotions using facial recognition and suggests music accordingly.

## Features
- Detects user emotions from facial expressions
- Recommends Spotify playlists based on emotion
- Uses AI-based emotion detection model
- Web interface for interaction

## Technologies Used
- Python
- Flask
- OpenCV
- HTML, CSS, JavaScript
- Spotify API

## How to Run the Project

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py
